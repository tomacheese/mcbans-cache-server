tomacheese/mcbans-cache-server

WebAPI that caches and distributes MCBans punishment information, player information, and server information.

End Points

Get Ban

  • GET /ban/{banId}

Get a ban by banId.

Get Player

  • GET /player/{playerId}
  • GET /player/{playerUUID}

Get a player by playerId or playerUUID.
UUID can be used either, whether there is hyphen.

Get Server

  • GET /server/{serverId}
  • GET /server/{serverAddress}

Get a server by serverId or serverAddress.

Get Stats

  • GET /stats

Get stats about the cache server.

Rate limit

Requests are limited to 100 times per minute.
When the rate limit is restricted, the relevant information is included in the response header Ratelimit-*.
